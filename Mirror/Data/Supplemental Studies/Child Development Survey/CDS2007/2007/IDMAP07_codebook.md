PCG68ID07 "PCG 1968 INTERVIEW NUMBER"              NUM(4.0)
PCG 1968 Interview Number (1968 ID Number)

This variable is the l968 family ID number for the CDS-2007 PCG. The combination of this
variable and Person Number (PCGPN07) provides unique identification for each individual on
the data file. Individuals associated with families from the Latino and Immigrant samples,
added to the PSID in 1990/1992 and 1997/1999, respectively, were assigned 1968 IDs so that
they, too, have unique identifiers compatible with the PSID structure for the core (SRC
and Census) sample.
